### 概述
- 该代码在参考坐标系上定义了**沿纤维方向的时空变化张力**并将其映射为第一 Piola–Kirchhoff 型应力张量 `PP`。  
- 张力沿体长产生行进波，且上下表面张力相位相反，从而可驱动弯曲或推进行为。  
- 最终结果 `PP` 可用于计算 Lagrangian 内力或通过离散算子得到作用在流体/结构上的力。

### 关键变量说明
- **`T0`**：张力基准幅值。  
- **`body_length`**：体长，用于归一化 x 方向位置。  
- **`x_min`**：参考体在 x 方向的起始位置，用于计算归一化坐标。  
- **`phase`**：局部相位，随位置和时间变化，定义为 `2π((X(0)-x_min)/body_length + time)`。  
- **`T_bottom` / `T_top`**：下表面与上表面的标量张力，二者相位差 π。  
- **`T`**：当前点使用的标量张力，按 `X(1) < 0` 选择上下表面。  
- **`FF`**：形变梯度张量 \(\mathbf{F}\)。  
- **`J`**：形变梯度的行列式 \(J=\det(\mathbf{F})\)。  
- **`f_0`**：参考纤维方向向量。  
- **`f_f`**：纤维外积 \(\mathbf{f}_0\otimes\mathbf{f}_0\)。  
- **`PP`**：计算得到的第一 Piola–Kirchhoff 型张量。

### 相位与张力计算细节
- **相位定义**：  
  \[
  \text{phase} = 2\pi\left(\frac{X(0)-x_{\min}}{\text{body\_length}} + \text{time}\right)
  \]
  相位随 \(x\) 增大而前移，随时间推进形成行进波。  
- **上下表面张力**：  
  \[
  T_{\text{bottom}} = \tfrac{1}{2}T_0\bigl(1+\sin(\text{phase})\bigr)
  \]
  \[
  T_{\text{top}} = \tfrac{1}{2}T_0\bigl(1+\sin(\text{phase}+\pi)\bigr)
  \]
  上下表面相位相反，导致在同一时刻上下两侧张力互为峰谷。  
- **按纵坐标选择**：若 \(X(1)<0\) 则使用 \(T_{\text{bottom}}\)，否则使用 \(T_{\text{top}}\)。

### 形变映射与张量构造
- 先计算纤维方向外积：  
  \[
  \mathbf{f}_f = \mathbf{f}_0 \otimes \mathbf{f}_0
  \]
- 构造第一 Piola–Kirchhoff 张量：  
  \[
  \mathbf{P} = J\,T\,\mathbf{F}\,\mathbf{f}_f
  \]
  等价写法：  
  \[
  \mathbf{P} = J\,T\,(\mathbf{F}\mathbf{f}_0)\otimes\mathbf{f}_0
  \]
- 物理含义：先将参考纤维方向通过 \(\mathbf{F}\) 映射到當前構型，再与参考纤维方向做外积并乘以标量张力，得到对参考配置的应力贡献。

### 数值与实现注意事项
- **纤维向量归一化**：确保 `f_0` 为单位向量以保持张力物理意义。  
- **检查 J 的值**：若 \(J\le 0\) 表示网格翻转或不可逆变形，应捕获并处理以避免数值崩溃。  
- **跨中性面平滑**：在 \(y=0\) 处上下张力直接切换可能引入不连续，建议使用平滑过渡以减少数值振荡。  
- **时间步长与刚性**：若 `T0` 或张力变化速率过大，会引入数值刚性，可能需要减小时间步长或采用隐式积分。  
- **边界处理**：体长端点处的相位或周期性条件应与整体模型一致，避免端点不连续。

### 建议改进与扩展
- **平滑切换**：在上下表面之间用连续函数（例如基于 \(X(1)\) 的平滑窗）替代硬阈值选择。  
- **相位控制**：将相位传播速度作为独立参数以控制波速和频率。  
- **稳定性保护**：在计算 `PP` 前加入对 \(J\) 的下限截断并记录异常点以便调试。  
- **可视化调试**：输出 `T`、`J` 与 `\|\mathbf{P}\|` 的剖面图以验证张力分布与应力映射是否符合预期。
